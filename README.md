# Boîte à outils du militant — Syndicat CGT

Application web syndicale : fiches juridiques, outils pratiques et modules de
formation pour les élu·es CSE (comité social et économique) et membres CSSCT
(commission santé, sécurité et conditions de travail).

## Accéder au site

**https://cgt-alfi.fr**

Le site est servi par GitHub Pages directement depuis [`index.html`](index.html)
à la racine du dépôt, sous le nom de domaine personnalisé déclaré dans
[`CNAME`](CNAME). Aucun build ni serveur requis.

## Contenu

| Rubrique | Description |
|----------|-------------|
| Mes droits face à l'enquête | Témoignage, preuve, prescription, contentieux |
| La BDESE | Base de données économiques, sociales et environnementales |
| Outils d'action CSE/CSSCT | Alertes, DUERP, expertises, stratégie graduée |
| Formations & Mandats | SSCT, droit d'alerte, AT/MP, RPS, outils pratiques |
